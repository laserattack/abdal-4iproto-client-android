# What is it

SSH-based VPN for Android. Device-wide tunneling, no root required

## Features

- Full device VPN over SSH
- Kill Switch
- Private DNS
- Split tunneling
- Auto-reconnect
- Multiple servers
- No telemetry

## Build

```
./gradlew clean assemble --no-daemon
```
